# Intro To DOM

Create JS objects that model the data contained in the HTML below

1.

  ```html
  <h1> A title </h1>
  <p> Some text </p>
  <p> Some more text </p>
  ```
2.

  ```html
  <div id="apple">
    <h1> Banana Title </h1>
    <p>Apple text</p>
  </div>
  ```
3.

  ```html
  <div id="biz">
    <img src="biz.png" />
    <div id="baz">
      <div id="buz">
        <table>
          <tr>
            <td>a</td>
            <td>b</td>
          </tr>
          <tr>
            <td>c</td>
            <td>d</td>
          </tr>
        </table>
      </div>
    </div>
  </div>
  ```
